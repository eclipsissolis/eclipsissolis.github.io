# Код узора — цифровой экспонат «Чеченский орнамент»

Интерактивный цифровой экспонат для Современного музея ЧГУ: посетитель складывает
собственный орнамент из подлинных знаков чеченского орнамента, выбирает форму
(круглая розетка или квадратное панно), меняет симметрию, цвет и отдельные элементы —
и уносит готовый узор как обои на телефон.

## Как открыть

Это один самодостаточный HTML-файл без зависимостей и без интернета.
Просто откройте **`index.html`** двойным кликом в любом браузере.

## Возможности

- 12 подлинных знаков: малх (солнце), беттаса (луна), седа (звезда), йиъ маьӀе
  (четыре стороны), дар, маӀаш (бараньи рога), хоро маӀаш (соединение родов),
  малхтӀемаш (солнце-крылья), наша ха (древо жизни), кхо патар (трилистник),
  нарт (арка), шотли (щит) — с названиями и смыслами.
- Две формы на выбор: **Круг** (радиальная розетка, симметрия 6/8/12) и
  **Квадрат** (осевое панно — древо жизни, рамки, бордюры, медальоны).
- **Кайма** — традиционный бегущий бордюр по краю (характерная черта чеченского
  орнамента): пять стилей на выбор — волна-завиток, лоза с лилиями (ковровая
  кайма), крылья (симметричный двойной завиток), рога (маӀаш), трилистник
  (кхо патар) — для обеих форм; узор рисуется внутри каймы.
- Готовые эскизы, кнопка «Случайный узор», редактирование отдельных элементов,
  экспорт готового узора в PNG, интерфейс РУС/ENG.
- Киоск-режим: после 90 секунд без касаний экспонат сам возвращается на заставку,
  стирает узор предыдущего посетителя и сбрасывает язык на русский.

## Источник знаков

Начертания и смыслы знаков воспроизведены по визуальному исследованию
**М. Гуноевой «Чеченский орнамент: история, форма, развитие»** (НИУ ВШЭ, 2023),
опирающемуся на материалы Л. Ильясова («Петроглифы Чечни») и Т. Шавлаевой.
На этапе внедрения библиотеку знаков следует выверить с историками ЧГУ.

## Технологии

Чистый HTML + Canvas 2D, без сборки и внешних библиотек. Иконки знаков и превью
эскизов «впечены» в файл как изображения, чтобы стабильно отображаться в любых
средах просмотра.
